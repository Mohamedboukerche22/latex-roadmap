# Start from Scratch :)
## You should memorize those commands to learn LATEX
<div align="center">

### Special Characters
 Fractions: `\frac{a}{b}` → $\frac{a}{b}$
 
 Square root: `\sqrt{x}` → $\sqrt{x}$
 
 Exponents: `x^2` → $x^2$
 
 Subscripts: `x_i` → $x_i$
 
  Sum: `\sum_{i=1}^n` → $\sum_{i=1}^n$
  
 Integral: `\int_a^b` → $\int_a^b$


| Code | Example|
|------|--------|
|`\frac`|$$\frac{x}{15}$$|
|`\sqrt`|$$\sqrt{150}$$|
|`\sum`|$$\sum_{i = 0}^{16}{i^2}$$|
|`5^2`|$$5^2$$|
|`x_2`|$$x_2$$|
|`\boxed{x}`|$$\boxed{x}$$ box|
|`\int(x)dx`|$$\int_ {1}^{2}{(x^2+16x)dx}$$|
|`\ln(x)`|$$\ln(x)$$|
